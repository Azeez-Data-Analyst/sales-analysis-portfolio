# 📊 Sales Performance Analysis Dashboard

## 📌 Project Overview
This project analyzes sales performance across regions, products, and sales representatives to identify key growth opportunities.

## 🎯 Objectives
- Analyze revenue trends
- Identify top-performing regions and products
- Evaluate sales team performance

## 🛠 Tools Used
- Microsoft Excel
- Power BI

## 📂 Dataset
Company sales dataset (anonymized)

## 📊 Key Insights
- Region North generated 45% of total revenue
- Vitamilk (Double-Choco SKU) showed a 20% decline over 3 months
- Sales increased by 15% in Q3

## 💡 Recommendations
- Increase focus on high-performing regions
- Improve strategy for declining products
- Train underperforming sales reps
